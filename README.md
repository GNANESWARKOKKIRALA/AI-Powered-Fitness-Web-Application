# 💪 Gnaneswar Fitness AI

AI-Powered Fitness Web Application using Python, Streamlit, Groq API, LLaMA 3.1 and MySQL.

## 🚀 Setup

### Step 1 — Install libraries:
```
pip install -r requirements.txt
```

### Step 2 — Create MySQL Database:
```sql
CREATE DATABASE fitnessai;
```

### Step 3 — Run the app:
```
streamlit run app.py
```

## 🛠️ Tech Stack
- Python | Streamlit | Groq API | LLaMA 3.1
- MySQL | bcrypt | Pandas | Plotly

## ✨ Features
- 🔐 Secure login with bcrypt
- 🥗 AI 7-day Indian diet plan
- 🏋️ AI Push Pull Legs workout
- 🤖 AI trainer chatbot
- 📈 Progress tracker with charts
